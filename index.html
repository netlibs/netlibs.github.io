The easiest way to handle projects with dependencies between projects is using prezi pride. Once you've installed it:
mkdir netlibs
pride init
pride add https://github.com/netlibs/io.netlibs.bgp.git
pride add https://github.com/netlibs/io.netlibs.ipaddr.git
pride add https://github.com/netlibs/io.netlibs.flow.git
You can then build with all dependencies:
gradle assemble
